# 计算简史 | Computer History Timeline

从算盘到 AI 的计算机发展史时间线网站。

![Preview](https://upload.wikimedia.org/wikipedia/commons/thumb/1/1e/Suanpan%281%29.jpg/800px-Suanpan%281%29.jpg)

## 简介

这是一个交互式的计算机发展史时间线网站，展示了从古代计算工具到现代人工智能的完整发展历程。

## 特色

- 🎨 赛博朋克风格的视觉设计
- 📱 响应式布局，支持移动端
- 🖼️ 重要里程碑配有详细图文介绍
- 🔍 左侧导航栏快速定位各个时代
- ✨ 流畅的滚动动画和交互动效

## 时间线涵盖

- **人工计算时代** (史前 - 1800): 算盘、算筹、对数表、计算尺、机械计算器
- **机械计算时代** (1822-1938): 差分机、图灵机
- **电子计算时代** (1940-1959): ENIAC、晶体管、集成电路
- **集成电路时代** (1960-1969): ARPANET
- **个人电脑时代** (1970-1979): Apple I
- **图形界面时代** (1980-1989): Macintosh
- **互联网时代** (1990-1999): WWW、深蓝
- **移动互联网** (2000-2009): iPhone
- **AI 复兴** (2010-2019): AlexNet、AlphaGo
- **大模型时代** (2020-至今): GPT-3、ChatGPT
